# 旅行规划

中文旅行规划 PWA 应用，支持行程管理、打包清单和预算追踪。

## 功能

- **行程管理** — 按日期分组展示行程项，支持增删改
- **打包清单** — 分类管理，带完成进度条
- **预算追踪** — 分类预算 + 支出记录

## 技术栈

React 18 · Vite 5 · localStorage · PWA

## 开发

```bash
npm install
npm run dev        # 启动开发服务器
npm run build      # 生产构建
npm run preview    # 预览生产构建
```

手机访问需加 `--host`：`npx vite --host`
